# Welcome to CryptoSentinel Framework

**A Professional Cryptographic Toolkit for Learning and Security Analysis**

---

## üéØ Overview

CryptoSentinel is a comprehensive Python framework that brings together classical cryptography, modern hashing algorithms, and security analysis tools into a single, easy-to-use application. Whether you're a student learning about cryptography, a security professional analyzing password strength, or a developer needing file integrity verification, CryptoSentinel provides the tools you need.

Built with production-quality code, extensive documentation, and a beautiful command-line interface, CryptoSentinel makes cryptography accessible to everyone while maintaining professional standards.

CryptoSentinel includes **10 powerful cryptographic tools**, each designed to solve specific security and cryptography challenges:

### üîê Classical Cipher Tools

* **[[Caesar-Cipher-Tool]]** - Simple shift cipher with automatic cryptanalysis using frequency analysis
* **[[Vigenere-Cipher-Tool]]** - Polyalphabetic substitution cipher with Kasiski examination and IoC analysis
* **[[Simple-Substitution-Cracker]]** - Monoalphabetic substitution cipher breaker using hill climbing AI algorithm
* **[[Simple-XOR-Encryption]]** - Bitwise XOR encryption with key length detection and analysis

### üì° Encoding Tools

### üîí Hashing & Integrity Tools

* **[[MD5-Hash-Calculator]]** - MD5 hash generation with collision vulnerability warnings
* **[[SHA256-Hash-Implementation]]** - Secure SHA-256 hashing with streaming support for large files
* **[[File-Checksum-Validator]]** - File integrity verification and corruption detection

### üõ°Ô∏è Security Analysis Tools

* **[[Password-Strength-Analyzer]]** - Password entropy calculator with crack-time estimation

## üöÄ Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/Shiva-destroyer/CryptoSentinel.git
cd CryptoSentinel

# Install dependencies
pip install -r requirements.txt

# Optional: Install as package
pip install -e .
```

### Launch the CLI

```bash
python cli.py
```

The interactive CLI will guide you through selecting and using any of the 10 tools.

## üéì Learning Path

**For Beginners:**
1. Start with [[Caesar-Cipher-Tool]] to understand basic cryptography
2. Progress to [[Vigenere-Cipher-Tool]] for polyalphabetic ciphers
3. Learn about hashing with [[MD5-Hash-Calculator]] and [[SHA256-Hash-Implementation]]
4. Understand security with [[Password-Strength-Analyzer]]

**For Security Professionals:**
1. Use [[Password-Strength-Analyzer]] for password policy validation
2. Verify file integrity with [[File-Checksum-Validator]]
3. Analyze encrypted data with cryptanalysis tools
4. Test encryption implementations with [[Simple-XOR-Encryption]]
